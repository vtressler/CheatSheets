HTML Cheat Sheet

HTML Tags

Tags Name Description
<html> html used to give the page and browsers instructions
<body> body used to designate the main body of the page
<title> title contains the title info of page; displayed in browser title bar
<head> head give info about the page that isn't displayed on the page
<!doctype html> doctype used to declare the document as html
<!--comment> comments used to write comments that aren't displayed by the browser
<div> dividers divides page into sections
<emphasize> em emphasizes text and will usually italicize the text in brackets
<br> break line break will create a break in text in a <div> element
</tagname> closing tag used to show the end of an html element
<elementname name=value> element name can be added to the opening tag to configure or change element behavior
<p> paragraph contains and displays a block of text
<h1-h6> headings 1-6 uses a level of 1 being highest 6 being lowest
<h1 id="A1"> Unique ID Attributes used to differentiate between elements
<img=src'path/to/image' alt="text description of img"/> alt alt attribute gives text descriptions
<span> span can be used to group for styling purposes, should be avoided for more specific elements
<strong> strong highlights important or serious text, will default to bold text
<a> anchor anchors hyperlinks to other pages or files
target= target attribute will tell the browser where to open hyperlinks
target="_blank" target in new will open a target in a new tab
# id="section" hyperlink within used to hyperlink to different parts of html documents
<table> table creates a table
<tr> table row adds table rows
<td> table data add table data
<thead> table head defines the headings of the table
<th> table heading used to add headings to rows and columns of a table
<tbody> table body element that will contain all table data
<tfoot> table footer uses table rows to give footer content or to sumarize content at the end of a table
rowspan row span indicates how many rows the cell takes up. It is set to 1 by default
colspan column span indicates how many columns the cell takes up. It is set to 1 by default..
src source contains the url and is mandatory for <img> and <video> elements
<embed> embed can be used to embed any media, must have a src element and is self-closing.
<video> video embeds a media player for a video playback must have a src attribute and closing tag
<img> image embeds an image on the page, must have a src element but no closing tag.
<audio> audio used to put audio on the page,must have a src element and closing tag.
<figure> figure is used to encapsulate media
<figcapture> figure capture is used to gice media a description
<section> section used to define chapters or headings, can be used to give theme
<article> article used to hold content that makes sense on its own, can be used to give theme content
<aside> aside marks additional data not needed to make sense of main contnet. ie sidebar
